## Getting Started
### Prerequisites
* npm
```
npm install npm@latest -g
```

### Installation
1. Clone the repo
```
git clone https://github.com/drew18moore/sorting-visualizer-react.git
```
2. Install npm packages
```
npm install
```

### Run
```
npm start
```
